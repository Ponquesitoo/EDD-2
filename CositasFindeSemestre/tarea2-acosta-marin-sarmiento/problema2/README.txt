Instrucciones para compilar:

El programa consiste en una cabecera, «TDA.hpp», su implementación, «funciones.cpp», y el programa principal, «problema2.cpp».
    - TDA.hpp contiene el TDA de árbol de búsqueda.
    - funciones.cpp implementa los métodos de la clase.
    - problema2.cpp contiene el main.

Para compilar el programa se debe navegar hasta la carpeta "/TAREA 2/problema 2" e ingresar el siguente comando:
    g++ problema1.cpp -Wall -o <nombre-ejecutable>

Y para la posterior ejecución, se debe utilizar el siguiente comando:
    ./<nombre-ejecutable>

Al ejecutarlo, se debe ingresar en la consola la instrucción que se desee ejecutar (ENCENDER i, CUANTOS_ENCENDER i o PARAR_PROGRAMA), siendo i un elemento entero.

El programa fue compilado en Ubuntu mediante WSL.
