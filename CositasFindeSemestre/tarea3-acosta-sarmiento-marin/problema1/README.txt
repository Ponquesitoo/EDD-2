Instrucciónes de Compilación

El código consiste en tres códigos, dos de ellos siendo cabeceras, mientras que el restante corresponde al main:
    - tda.hpp : contiene el TDA Login, junto con la implemetación de las funciones método
    - hash.hpp : contiene el TDA Hashing, junto con la  implementación de los métodos de la clase
    - problema1.cpp : almacena el main para ejecutar el programa 

Para compilar el programa se debe navegar hasta la carpeta "/tarea3/problema1" e ingresar el siguente comando:
    g++ problema1.cpp -Wall -o <problema1-ejecutable>

Y para la posterior ejecución, se debe utilizar el siguiente comando:
    ./<problema1-ejecutable>
Para posteriormente ingresar las instrucciones por medio de la entrada estándar.

Finalmente, el programa fue compilado en Ubuntu mediante WSL.
