Instrucciones de Compilacion

El codigo consiste de dos secciones: Parte 1 y Parte 2.

Para compilar el programa se debe navegar hasta la carpeta "/tarea1-acosta-sarmiento" e ingresar los siguentes comandos:

Primero se debe compilar la seccion 1, por tanto: g++ parte1.cpp -Wall -o

Y para la posterior ejecucion, se debe utilizar el siguiente comando: ./

Lo mismo para la siguiente parte del programa g++ parte2.cpp -Wall -o

Y para la posterior ejecucion, se debe utilizar el siguiente comando: ./

Para posteriormente ingresar las instrucciones por medio de la entrada estandar.
